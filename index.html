2.1 Derivatans definition

▼

Derivatan mäter hur snabbt en funktion förändras. Tänk: om f(x) är position, så är f'(x) hastigheten. Grafiskt = lutningen på tangentlinjen i en punkt. Positiv derivata → funktionen ökar. Negativ derivata → den minskar. Noll → vågrät tangent.

Tangentlinjen i x = 2 har lutning f'(2) = 4

💡 Tumregel: f'(x) berättar om grafen lutar uppåt (+), nedåt (−) eller är platt (0) i varje punkt.
f'(x) = limh→0 [f(x+h) − f(x)] / h f'(x) > 0 → f växer f'(x) < 0 → f avtar f'(x) = 0 → vågrät tangent 📄 Formelblad
▶ Sök genomgång på YouTube
Genomgångsexempel

Fråga: f(x) = x2. Beräkna f'(3) med definitionen.

f'(3) = limh→0 [(3+h)2 − 9] / h
= limh→0 [9 + 6h + h2 − 9] / h
= limh→0 [6h + h2] / h
= limh→0 (6 + h)
= 6

Tangentens lutning i x = 3 är alltså 6.

2.2 Potensregeln & tangenter

▼

Det här är motorn i derivering. Potensregeln gör att du kan derivera alla polynom snabbt. Sedan kan du skriva tangentens ekvation i vilken punkt som helst. Allt du behöver kunna: flytta ner exponenten, minska den med 1.

💡 Mönstret: "Flytta ner exponenten framför, minska exponenten med 1." Konstanter försvinner. Derivera varje term för sig.
f(x) = xn → f'(x) = n · xn−1 f(x) = k · xn → f'(x) = k · n · xn−1 Konstant → derivata = 0 Tangent i x = a: y = f'(a)(x − a) + f(a) 📄 Formelblad
▶ Sök genomgång på YouTube
Genomgångsexempel

Fråga: f(x) = 3x4 − 2x2 + 7x − 5. Bestäm f'(x) och skriv tangentens ekvation i x = 1.

Derivera:
3x4 → 3·4·x3 = 12x3
−2x2 → −2·2·x1 = −4x
7x → 7
−5 → 0
f'(x) = 12x3 − 4x + 7
Tangent i x = 1:
f(1) = 3 − 2 + 7 − 5 = 3
f'(1) = 12 − 4 + 7 = 15
y = 15(x − 1) + 3 = 15x − 12

2.3 e, ln & exponentialfunktioner

▼

Talet e (≈ 2,718) är speciellt: ex är sin egen derivata. Det gör exponentialfunktioner enkla att derivera. ln (naturliga logaritmen) är inversen till e — du behöver den för att lösa ekvationer med e. Alla andra baser (t.ex. 2x) går via ln.

💡 Mönstret: enåt → samma grej × derivatan av "nåt". Har du annan bas ax? Multiplicera med ln(a).
f(x) = ex → f'(x) = ex f(x) = ekx → f'(x) = k · ekx (kex-regeln) f(x) = C · ekx → f'(x) = C · k · ekx f(x) = ax → f'(x) = ax · ln(a) ln(ex) = x   |   eln(x) = x 📄 Formelblad
▶ Sök genomgång på YouTube
Genomgångsexempel

Fråga 1: Derivera f(x) = 5e3x

Kex-regeln: koefficient × k × ekx
f'(x) = 5 · 3 · e3x = 15e3x

Fråga 2: Lös e2x = 10

Ta ln av båda led: ln(e2x) = ln(10)
2x = ln(10)
x = ln(10)/2 ≈ 1,15

3.1–3.2 Extremvärden Bonus

▼

Här använder du derivatan för att hitta max och min. Sätt f'(x) = 0, kolla om derivatan byter tecken. Andraderivatan f''(x) berättar om krökningen. Inflexionspunkt = där krökningen byter riktning.

f'(x) = 0 vid max- och minpunkterna. f' byter tecken.

💡 Receptet: 1) Derivera → 2) Sätt f'(x) = 0 → 3) Teckentabell för f' → 4) Plusminus-byte = extrempunkt. Plus→minus = max, minus→plus = min.
Extrempunkt: f'(x) = 0 och f' byter tecken Max: f' går från + till − Min: f' går från − till + Inflexionspunkt: f''(x) = 0 och f'' byter tecken Slutet intervall [a,b]: jämför f(a), f(b) och alla f(x₀) där f'(x₀) = 0 📄 Formelblad
▶ Sök genomgång på YouTube
Genomgångsexempel

Fråga: Bestäm extrempunkterna för f(x) = x3 − 3x.

f'(x) = 3x2 − 3
f'(x) = 0: 3x2 − 3 = 0 → x2 = 1 → x = ±1
Teckentabell:
x < −1: f'(−2) = 12 − 3 = 9 > 0 (växande)
−1 < x < 1: f'(0) = −3 < 0 (avtagande)
x > 1: f'(2) = 12 − 3 = 9 > 0 (växande)
x = −1: max (+ → −), f(−1) = 2 → maxpunkt (−1, 2)
x = 1: min (− → +), f(1) = −2 → minpunkt (1, −2)

Vad är en integral?

▼

Integralen beräknar arean mellan grafen och x-axeln. Area ovanför x-axeln räknas som positiv, area under x-axeln som negativ. Om det finns lika mycket ovan som under → integralen = 0. OBS: Arean själv är alltid positiv, men integralvärdet kan bli negativt.

Grön yta = positivt bidrag. Röd yta = negativt bidrag. Integralen = nettot.

💡 Kom ihåg: Integral = "nettoarean" (med tecken). Vill du ha arean? Ta absolutbeloppet av varje del.
∫ab f(x) dx = arean med tecken Ovanför x-axeln → positivt bidrag Under x-axeln → negativt bidrag 📄 Formelblad
▶ Sök genomgång på YouTube
Genomgångsexempel

Tänk dig: f(x) = 2 (en horisontell linje) från x = 0 till x = 3.

Arean = en rektangel med bredd 3 och höjd 2
∫03 2 dx = 2 · 3 = 6

Om f(x) = −1 från x = 0 till x = 4:

Grafen ligger under x-axeln
∫04 (−1) dx = −1 · 4 = −4 (negativt!)
Men arean är |−4| = 4.

Integralnotation

▼

Integralnotationen ser skrämmande ut men har bara fyra delar. Lär dig läsa den som en mening: "Beräkna arean under f(x) från a till b."

💡 Läs det som: ∫ = "summan av", a/b = "från/till", f(x) = "funktionen", dx = "med avseende på x"
∫ab f(x) dx ∫ = integrationstecknet (ett utdraget S) a = undre gräns (start-x) b = övre gräns (slut-x) f(x) = integranden (funktionen) dx = "med avseende på x" (ska alltid vara med)

Uppskatta integraler

▼

Ibland ska du uppskatta en integral från en graf utan att räkna. Räkna rutor! Kolla skalan noggrant — en ruta kanske inte är 1×1. Trianglar, rektanglar och halvcirklar har kända formler.

Räkna rutorna under kurvan. Varje ruta = 1 areaenhet. Halva rutor ≈ 0,5.

💡 Strategi: Dela upp området i kända former (rektanglar, trianglar, halvcirklar). Kolla axlarnas skalning! Under x-axeln → minus.
Rektangel: bas × höjd Triangel: bas × höjd / 2 Halvcirkel: πr² / 2 Under x-axeln → negativt bidrag

Primitiva funktioner

▼

Primitiv funktion = omvändningen av derivering. Istället för att "ta ner exponenten" → höj exponenten med 1 och dela. Glöm inte konstanten +C vid obestämda integraler (utan gränser).

💡 Mönstret: Derivering baklänges: höj exponenten med 1, dela med den nya exponenten. Plus C!
∫ xn dx = xn+1/(n+1) + C ∫ 3x2 dx = 3 · x3/3 = x3 + C ∫ k dx = kx + C ∫ ekx dx = ekx/k + C Glöm inte +C! (vid obestämd integral) 📄 Formelblad
▶ Sök genomgång på YouTube
Genomgångsexempel

Fråga: Bestäm den primitiva funktionen till f(x) = 4x3 − 6x + 2.

4x3 → 4 · x4/4 = x4
−6x → −6 · x2/2 = −3x2
2 → 2x
F(x) = x4 − 3x2 + 2x + C

Med villkor: Om F(1) = 5, bestäm C.

F(1) = 1 − 3 + 2 + C = C = 5
C = 5, så F(x) = x4 − 3x2 + 2x + 5

Bestämd integral — steg för steg

▼

Det här är den viktigaste metoden att kunna. Varje steg ska skrivas tydligt — det ger poäng på provet! Följ receptet nedan exakt, så blir det alltid rätt.

∫ₐᵇ f(x)dx = arean under kurvan mellan a och b

💡 7-stegsreceptet: Skriv integralen → primitiv i hakparentes → stoppa in övre gränsen − undre gränsen → räkna ut → svara.
∫ab f(x) dx = [F(x)]ab = F(b) − F(a) 📄 Formelblad
▶ Sök genomgång på YouTube
Genomgångsexempel (från lektionen)

Beräkna ∫−23 (3x2 + 1) dx

1. Skriv integralen: ∫−23 (3x2 + 1) dx
2. Bestäm primitiven: 3x2 → x3, och 1 → x
3. Skriv med hakparentes: = [x3 + x]−23
4. Stoppa in övre gränsen (3): [33 + 3]
5. Stoppa in undre gränsen (−2): [(−2)3 + (−2)]
6. Räkna ut varje parentes: [27 + 3] − [−8 + (−2)] = 30 − (−10)
7. = 30 + 10 = 40

Area: under x-axeln & mellan kurvor

▼

När du ska beräkna riktig area (inte bara integralen) måste du vara försiktig med delar under x-axeln. Dela upp i intervall, ta absolutbelopp av varje del. Mellan kurvor: integrera |f(x) − g(x)|.

Arean mellan f(x) och g(x) = ∫ |f(x) − g(x)| dx

💡 Strategi: 1) Hitta var grafen korsar x-axeln (nollställen). 2) Integrera varje bit separat. 3) Ta absolutbelopp av negativa delar. 4) Addera.
Area under x-axeln: ta |∫ab f(x) dx| Area mellan f och g: ∫ab |f(x) − g(x)| dx Om f(x) ≥ g(x) på [a,b]: ∫ab [f(x) − g(x)] dx
▶ Sök genomgång på YouTube
Genomgångsexempel

Fråga: Beräkna arean mellan f(x) = x2 och x-axeln på [−1, 2] om f(x) ≥ 0 på hela intervallet.

Kolla: x2 ≥ 0 alltid → hela grafen ovanför x-axeln
Area = ∫−12 x2 dx = [x3/3]−12
= 8/3 − (−1/3) = 8/3 + 1/3
= 9/3 = 3

Area mellan kurvor: f(x) = x2 och g(x) = x från 0 till 1.

Kolla: på [0,1] gäller x ≥ x2, dvs g(x) ≥ f(x)
Area = ∫01 (x − x2) dx = [x2/2 − x3/3]01
= (1/2 − 1/3) − (0) = 3/6 − 2/6
= 1/6
